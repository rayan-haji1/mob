دروست کردنی پەیجى فەیسبوک:-
سلاو ئازیزان و خۆشەویستان لە بابەت و فێرکاری ئەم جارەماندا باس لە دانانی پەیجی فیسبوک دەکەین سەرەتا دەچینە سەر بەشی رێکخستنە سەرەکیەکان پاشان کلیک لە page دەکەین پاشان بەشێکمان بۆ ئەبێتەوە پاشان کلیک ☟ Create دەکەین پاشان کلیک لە☟ Get Started دەکەین پاشان لە page name ناوی پەیجەکەت دەنوسیت و کلیک لە Next دەکەیت پاشان بەشێکی ترمان بۆ دەکرێتەوە ئەڵێت دەتەوێ پەیجەکەت سەبارەت بە چی کارێک بێت کارەکەت هەڵدەبژێریت و پاشان کلیک لە Next دەکەیت پاشان بەشێکی ترمان بۆ دەکرێتەوە ئەگەر وێب سایتت هەبوو لەو بەشە دایدەنێی گەر نەتبوو بە بەتاڵی جێی بھێڵە ، ھەروەھا دەتوانی شوێن و شار و وڵات ھەڵبژێری واتا ناونیشانەکەت ئەگەر بتەوێت ... ھەروەھا ئەشتوانی ھەمووی بە بەتاڵی جێ بێڵیت دواتر بەشێکی دیکەمان بۆ دەکرێتەوە کە تایبەتە بە دانانی وێنەی کەڤەری پەیج و دانانی وێنەی سەرەکی پەیج Add profile picture واتا دانانی وێنە لەسەر پەیج Add photo واتا دانانی وێنە لەسەر ( کەڤەری پەیج و سەرەکی پەیجەکەش ھەمان شت دەدات ) پاشان وێنەیەک بەئارەزوی خۆت ھەڵدەبژێریت ( وێنەشی لەسەر دانەنێی ئاساییە) و کلیک لە Done بکە ئەگەر ھەموو ئەم ھەنگاوانەت جێبەجێ کرد ئەوا ئەبیتە خاوەن پەیجی تایبەت بە خۆت ھەروەھا دەتوانی ژمارە مۆبایل یاخود ئیمێڵ دانێیت، بۆ ئەوەی پەیوەندیت پێوە بکرێت(( ئەم زیاتر بۆ ئەوانەیە کە کاری کڕین و فرۆشتن دەکەن)) بەهیوای سوود ..
